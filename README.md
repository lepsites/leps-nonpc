# 🚫 leps-nonpc

[TR] FiveM sunucuları için tamamen optimize edilmiş, FPS arttıran gelişmiş trafik ve NPC (yaya/araç) kontrol scripti.
[EN] A fully optimized, FPS-boosting advanced traffic and NPC (ped/vehicle) control script for FiveM servers.

## 🌟 Features / Özellikler

*   **[TR]** 0.00ms Resmon değeri (Tamamen optimize) / **[EN]** 0.00ms Resmon usage (Fully optimized)
*   **[TR]** Trafik yoğunluğunu tamamen kapatma veya azaltma / **[EN]** Completely disable or reduce traffic density
*   **[TR]** Sokaklarda yürüyen yayaları (ped) kapatma / **[EN]** Disable walking pedestrians on the streets
*   **[TR]** Park halindeki araçları engelleme / **[EN]** Disable parked vehicles
*   **[TR]** Acil servis (Polis/Ambulans) NPC'lerini engelleme / **[EN]** Disable emergency services (Police/Ambulans) NPCs
*   **[TR]** Çöp kamyonu vb. rastgele servis araçlarını kapatma / **[EN]** Disable random service vehicles like garbage trucks

## 🛠️ Installation / Kurulum

1.  **[TR]** Dosyayı indirin ve `resources` klasörünüzün içine atın.
    **[EN]** Download the file and drop it into your `resources` folder.
2.  **[TR]** Klasör adının `leps-nonpc` olduğundan emin olun.
    **[EN]** Make sure the folder is named `leps-nonpc`.
3.  **[TR]** `server.cfg` dosyanıza aşağıdaki satırı ekleyin:
    **[EN]** Add the following line to your `server.cfg`:
    ```cfg
    ensure leps-nonpc
    ```

## ⚙️ Configuration / Ayarlar

**[TR]** Script içerisindeki ayar dosyasından (veya client dosyasının başından) yoğunluk değerlerini `0.0` (tamamen kapalı) ile `1.0` (varsayılan) arasında istediğiniz gibi değiştirebilirsiniz.
**[EN]** You can change the density values between `0.0` (completely disabled) and `1.0` (default) from the config file (or at the top of the client file) as you wish.
